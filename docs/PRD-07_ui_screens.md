## 3. 영역별 구성

### 3.1 상단 내비게이션 바

| 요소 | 기능 |
|------|------|
| 로고 | 홈으로 이동 |
| 설정(⚙) | 환경설정 (인코딩, 테마 등) |
| 계정(👤) | 로그인/프로필 (확장 기능) |

### 3.2 좌측 사이드바

| 1depth 메뉴 | 2depth 메뉴 | 기능 | 접근 권한 |
|-------------|-------------|------|----------|
| 우대사업자 | 사업자 조회 | 국세청 우대 등급 조회/설정 | 총판 |
| 대시보드 | - | 매출 현황, 순위, 알림 | 전체 |
| 영업점 관리 | 영업점 등록 | 신규 영업점 생성 | 총판, 대리점 |
|  | 영업점 목록 | 영업점 조회/수정/삭제 | 총판, 대리점 |
| 가맹점 관리 | 가맹점 등록 | 신규 가맹점 생성 | 대리점 이상 |
|  | 가맹점 목록 | 가맹점 조회/수정/삭제 | 대리점 이상 |
|  | 단말기 관리 | 단말기 등록/조회/관리 | 대리점 이상 |
| 정산 관리 | 영업점 정산내역 | 영업점별 정산 조회/확정 | 전체 |
|  | 가맹점 정산내역 | 가맹점별 정산 조회/확정 | 전체 |
| 지급 이체 | 지급이체 등록 | 정산금 송금 요청 | 총판 |
|  | 지급이체 조회 | 송금 내역 조회 | 총판 |
| 결제 관리 | 결제내역 | 거래 내역 조회 | 전체 (권한별 필터) |
|  | 실패내역 | 결제 실패 건 조회/분석 | 전체 (권한별 필터) |
| 운영 관리 | 공지사항 | 공지 등록/조회 | 총판 |
|  | 계정관리 | 사용자 계정 CRUD | 총판 |
|  | 환경설정 | 시스템 설정, 감사 로그 | 총판 |

### 3.3 좌측 메뉴별 기능 상세

| 1depth | 2depth | 주요 기능 |
|--------|--------|----------|

|  | 환경설정 | 개발연동 |

### 3.4 중앙 작업 영역

| 요소 | 기능 |
|------|------|
| 탭 바 | 열린 메뉴 탭 표시, 닫기/추가 버튼, 최대 10개 탭 지원 |
| 콘텐츠 영역 | 좌측 메뉴 선택 시 해당 화면 표시 |

- 탭은 독립적으로 운영됨

#### 3.4.1 탭 관리 기능

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ [대시보드] [가맹점 목록 ×] [영업점 상세 ×] [정산 내역 ×]    [✖ 전체 닫기]  │
└─────────────────────────────────────────────────────────────────────────────┘
```

**탭 바 기능**:

| 기능 | 설명 |
|------|------|
| 탭 클릭 | 해당 탭으로 포커스 이동 |
| × 버튼 | 개별 탭 닫기 (closeable 탭만) |
| 전체 닫기 버튼 | 탭 바 오른쪽 끝, 닫을 수 있는 모든 탭 닫기 |
| 드래그 & 드롭 | 탭 순서 변경 |

**우클릭 컨텍스트 메뉴**:

탭에서 마우스 우클릭 시 컨텍스트 메뉴 표시:

| 메뉴 항목 | 단축키 | 설명 |
|----------|--------|------|
| 탭 닫기 | - | 선택한 탭 닫기 (closeable 탭만 표시) |
| 다른 탭 닫기 | - | 선택한 탭을 제외한 나머지 닫기 |
| 전체 탭 닫기 | - | 닫을 수 있는 모든 탭 닫기 |

**탭 상태**:

| 속성 | 설명 |
|------|------|
| closeable: true | 닫기 가능 (× 버튼 표시) |
| closeable: false | 닫기 불가 (대시보드 등 고정 탭) |

> **Note**: 고정 탭(closeable: false)은 "다른 탭 닫기", "전체 탭 닫기" 실행 시에도 유지됩니다.

### 3.5 하단 패널

| 탭 | 기능 |
|-----|------|
| Console | 파싱 로그, 에러 메시지 |
| Diff | 두 전문 비교 (요청 vs 응답) |

## 4. 화면별 상세 설계

### 4.1 우대사업자 조회

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ 우대사업자 조회                                                              │
├─────────────────────────────────────────────────────────────────────────────┤
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 사업자번호 │                                            │ [조 회]  │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ ⚠️ 사업자번호 상태 (계속, 휴업, 폐업) 확인 불가합니다.               │   │
│  │ ⚠️ 사업자번호 다중 조회시 최대 10개 까지 "," 구분으로 가능합니다.    │   │
│  │    (Ex : 2068190716,3153400737)                                     │   │
│  └─────────────────────────────────────────────────────────────────────┘   │
│                                                                             │
│  ┌─────────────────────────────────────────────────────────────────────┐   │
│  │ 우대사업자 정보                                                      │   │
│  ├──────────────┬──────────────┬──────────────┬──────────────┐         │   │
│  │  사업자번호   │  영중소등급   │ 2025년 상반기 │ 2025년 하반기 │         │   │
│  ├──────────────┼──────────────┼──────────────┼──────────────┤         │   │
│  │ 2068190716   │     일반     │     일반     │     일반     │         │   │
│  ├──────────────┼──────────────┼──────────────┼──────────────┤         │   │
│  │ 3153400737   │     영세     │     영세     │     영세     │         │   │
│  └──────────────┴──────────────┴──────────────┴──────────────┘         │   │
│                                                                             │
└─────────────────────────────────────────────────────────────────────────────┘
```

**기능**:
- 우대사업자 조회 제공
- 최대 10개까지 ","으로 구분하여 조회 가능
- 사업자번호, 영중소등급, 연도별 등급 표기 (API 연동 추후 추가)

```
┌────────────┐
│ 사업자 정보 │
└────────────┘
                            ┌─────────────────┐
                            │ 법인 사업자   ▼ │
                            └─────────────────┘

┌─────────────────────────────────────┐ ┌─────────────────────────────────────┐
│ 상 호 :                             │ │ 대 표 자 :                          │
└─────────────────────────────────────┘ └─────────────────────────────────────┘

┌───────────────────────────┐┌──────────────────────┐ ┌────────────────────────────┐
│ 개업 연월일 :             ││ 사업자등록번호:        │ │ 법인등록번호 :            │
└───────────────────────────┘└──────────────────────┘ └───────────────────────────┘

┌───────────────┐ ┌───────────────────────────────────────────────────────────┐
│ 우편번호 :    │ │ 사업장 소재지 :                                           │
└───────────────┘ └───────────────────────────────────────────────────────────┘

                  ┌───────────────────────────────────────────────────────────┐
                  │ 실사업장 소재지 :                                         │
                  └───────────────────────────────────────────────────────────┘

┌───────────────────────┐ ┌───────────────────────┐ ┌─────────────────────────┐
│ 업태 :                │ │ 업종 :                │ │ 대표번호 :              │
└───────────────────────┘ └───────────────────────┘ └─────────────────────────┘

┌───────────────────────┐ ┌───────────────────────┐ ┌─────────────────────────┐
│ 담당자 :              │ │ 연락처 :              │ │ E-Mail :                │
└───────────────────────┘ └───────────────────────┘ └─────────────────────────┘
```

**사업자 유형별 필드**:

| 유형 | 필수 필드 | 선택 필드 |
|------|----------|----------|
| 법인사업자 | 사업자번호, 법인등록번호, 상호, 대표자, 주소 | 개업일, 업태, 업종, 연락처 |
| 개인사업자 | 사업자번호, 상호, 대표자, 주소 | 개업일, 업태, 업종, 연락처 |
| 비사업자 | 상호, 대표자, 주소 | 연락처 |

**사업자 검색 기능** (신규):
- 사업자번호 입력 후 검색 시 기존 등록 사업자 조회
- 기존 사업자 선택 → 자동 채움
- 미등록 사업자 → 새로 등록 모드

### 7.2 필터 기능

| 필터 | 옵션 | 설명 |
|------|------|------|
| 상태 | 전체, 승인, 취소, 부분취소 | 거래 상태별 필터링 |
| 날짜 유형 | 거래일자, 승인일자, 취소일자 | 검색 기준 날짜 선택 |
| 기간 | DateRangePicker | yyyy/MM/dd 형식, 달력 팝업 |
| 빠른 선택 | 7일, 30일, 90일 | 오늘 기준 과거 N일 자동 설정 |

### 7.3 상세검색

| 필드 | 설명 |
|------|------|
| 거래 TID | 거래 ID 부분 검색 |
| 승인번호 | 승인번호 부분 검색 |
| PG ID | PG 연결 ID 필터 |

### 7.4 날짜 유형별 필터링 규칙

| 날짜 유형 | 백엔드 동작 |
|----------|------------|
| 거래일자 (CREATED) | createdAt 기준 필터링 |
| 승인일자 (APPROVED) | approvedAt != null 필수, approvedAt 범위 필터링 |
| 취소일자 (CANCELLED) | cancelledAt != null 필수, cancelledAt 범위 필터링 |

> **중요**: 승인일자/취소일자 선택 시 해당 날짜 필드가 없는 건은 자동 제외됨

### 7.5 테이블 컬럼

| 컬럼 | 설명 | 정렬 |
|------|------|------|
| No. | 행 번호 (페이지별 연속) | - |
| 거래ID | transactionId | ✓ |
| 가맹점 | merchantName (가맹점명) | ✓ |
| 금액 | 통화 포맷 (₩) | ✓ |
| 상태 | Badge (승인/취소/부분취소) | ✓ |
| 승인일시 | yyyy-MM-dd HH:mm:ss | ✓ |
| 거래일시 | createdAt | ✓ |
| 상세 | 상세 버튼 | - |
